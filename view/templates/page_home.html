About

This is a May, 2016, data version-locked Linked Data distribution of the G-NAF, the Geocoded National Address File.

The main product in this series is the Linked Data distribution of the current G-NAF data, available at http://linked.data.gov.au/dataset/gnaf.

The purpose of this version is to present data as it was on the dya of the 2016 Census.

More information

This version of the G-NAF's Linked Data ditribution operates exactly the same as the current version. Please see that version's documentation for more information:
